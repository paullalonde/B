Tarabiscoter Sample

This sample application is a grab-bag showing various techniques and/or features of B.  The features demonstrated by the application are accessible from the Test menu.  They are enumerated below.

Drawers

Displays a window with three drawers, on the left, right and bottom edges.  A button on each edge toggles the visibility of corresponding drawer.  Code-wise, it's a simple use of the Drawer class.

Layout [Compositing]

Demonstrate the layout support in B.  By choosing from different popup menus on the left, you change the layout information for one of the bevel buttons on the right (the one labelled "Victim").

Layout [Non-Compositing]

Same as above, except the window is non-compositing.  The contents of both windows are identical.  It's a handy way of comparing the difference in behaviour between the two.

Split

Demonstrates SplitView, which is a class currently under development that manages two subviews and performs cursor tracking.

Modal Dialog

Demonstrates the AutoWindow and Dialog classes.  Displays an application-modal dialog.

FAQ

What is tarabiscoter?

Tarabiscoter is a French verb (pronounced "tah-rah-bees-coh-tay") which in its figurative sense means "to complicate unnecessarily".  It's an allusion to the fact that the application is a collection of contrived examples :)
