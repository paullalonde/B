DragPeekerX Sample

A re-creation for Mac OS X of a utility that used to come with the Macintosh Drag and Drop SDK.  It consists of a window that one can drag things into.  The window displays a dump of the drag's contents.

The main classes are described next.

Application

The main application class. This class derives from B::Application<B::NullDocumentPolicy, B::NullPrintPolicy>, which is a long way of saying that it's a documentless application (because its document policy is B::NullDocumentPolicy), and that it doesn't support printing (because its print policy is B::NullPrintPolicy).

The class is very simple.  The only function longer than a few lines is CreateDragPeekerWindow(), which creates a new drag peeker window.

DragPeekerView

This class is a custom view that displays the contents of a drag.  It demonstrates the following techniques:

  • Custom HIView development with the B framework.
  • Embeddability within an HIScrollView.
  • Responding to drag & drop actions.
  • Drawing with CoreGraphics.

System Requirements

  • Mac OS X 10.4 (aka Tiger).
  • One of :
    • Xcode 2.1
    • CodeWarrior 10.0
