Detailed Description

template<typename T>
class B::AutoTypedPtr< T >

Type-safe wrapper around Carbon Ptr type.

This class provides "resource acquisition is initialization" (RAII) semantics for Carbon Ptrs. It features a destructive copy constructor and assignment operator similar in spirit to std::auto_ptr.

It's assumed that the ptrs given to objects of this type match the template parameter T. T can be any POD (aka Plain Old Data) type.
