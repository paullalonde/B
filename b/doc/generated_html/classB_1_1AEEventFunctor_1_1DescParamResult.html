Detailed Description

template<DescType TYPE>
class B::AEEventFunctor::DescParamResult< TYPE >

Functor returns an Apple Event result typed as per DescParam.

This functor is meant to be passed to any argument of instantiations of AEEvent that return a result. The result is coerced if necessary to the C/C++ language type denoted by TYPE.
