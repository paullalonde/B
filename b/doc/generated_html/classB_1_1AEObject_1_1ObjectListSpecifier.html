Detailed Description

template<class INPUT_ITERATOR>
class B::AEObject::ObjectListSpecifier< INPUT_ITERATOR >

Writes out a list of object specifiers to an AEWriter.

The objects are given in the form of a pair of input iterators. The iterators are assumed to point to data of type AEObject* or const AEObject*. That is, dereferencing the iterators must yield one of the two types.
