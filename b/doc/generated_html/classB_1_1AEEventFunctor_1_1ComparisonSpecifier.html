Detailed Description

template<typename OBJ_SPEC, typename OBJ_CLASS, typename CMP_OBJ1, typename CMP_OBJ2, typename CMP_OP>
class B::AEEventFunctor::ComparisonSpecifier< OBJ_SPEC, OBJ_CLASS, CMP_OBJ1, CMP_OBJ2, CMP_OP >

Functor that holds an object specifier for a comparison.

This functor is meant to be passed to any of the arguments to instantiations of AEEvent that take an object specifier.
