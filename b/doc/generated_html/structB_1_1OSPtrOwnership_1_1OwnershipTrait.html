Detailed Description

Policy class for manipulating the refcount of an OS-managed data structure.

Although data structures managed by Mac OS X are almost always "CF" (i.e., CoreFoundation) objects nowadays, there remain a few data structures that have their own retain/release APIs.

OwnershipTrait is a "policy" class (i.e. it declares an API, but doesn't provide an implementation). It declares an API for manipulating the reference count for objects managed by the OS. This policy is implemented in a number of different classes (one per API obviously).
