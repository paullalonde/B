Detailed Description

Do-nothing print policy.

This class is a stub implementation of PRINT_POLICY. Most of its member functions do nothing.

This class can be used as a template parameter to those classes that take an PRINT_POLICY template parameter.

Currently, two classes take undo policies as template parameters: Application and Document.
