Detailed Description

Base class for custom derivatives of HIObject.

Once an EventTarget is instantiated, its lifetime is tied to that of its underlying HIObjectRef. Its lifetime should be controlled using the normal CF mechanisms (CFRetain & CFRelease) rather than operator delete.

template<class T >
void B::EventTarget::Register ( )
static

Registers class T (which must derive from EventTarget) with the HIToolbox.

The template parameter T must have the following characterstics:

  • It must contain a static member variable of type CFStringRef called kHIObjectClassID. This variable contains the class's unique identifier.
  • It may contain a static member variable of type CFStringRef called kHIObjectBaseClassID. This variable contains the base class's unique identifier. If this variable isn't present, a default value defined in EventTarget is used, in which case T is assumed to derived directly from HIObject.
  • It must contain a public static member function called Instantiate which has the same signature as ConstructProcPtr. The function needs to instantiate a T and return it as an EventTarget.

TTemplate parameter. Should be a class that derives from EventTarget.
