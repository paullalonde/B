Detailed Description

Client interface to the Undo mechanism.

All member functions take an EventTargetRef as an argument. Because implementations of UNDO_POLICY are attached to an EventTargetRef, this argument serves to direct requests to a given UNDO_POLICY implementation. By default, the request is sent to the user input focus, and travels up the handler chain from there.

See also
Supporting Undo, Undo, UNDO_POLICY

DEPRECATED. Adds the adapable unary functor inFunctor to the currently open action group.

FUNCTOR must be an adaptable unary function object. Its operator () must take one argument of type FUNCTOR::argument_type. Its result is discarded, so a return type of void is acceptable. The value of the argument is given in inArg.
