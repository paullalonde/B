Detailed Description

Functor that holds an optional "save option".

This functor is meant to be passed as the inSaveOption argument to AEEvent<kCoreEventClass, kAEQuitApplication>::Send() and AEEvent<kAECoreSuite, kAEClose>::Send().

A "save option" is an enumerated type whose possible values are kAEYes, kAENo and kAEAsk. It is only written out if its value is different from the default value of kAEAsk.
