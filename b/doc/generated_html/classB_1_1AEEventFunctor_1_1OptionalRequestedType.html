Detailed Description

Functor that holds an optional requested data type.

This functor is meant to be passed as the inRequestedType argument to AEEvent<kAECoreSuite, kAEGetData>::Send().

The requested data type is only written out if isn't either 0 or typeWildCard. Its default value is typeWildCard.
