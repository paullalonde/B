This is an overloaded member function, provided for convenience. It differs from the above function only in what argument(s) it accepts.

This function calls MOSL to perform the coercion. This means that if desc is an object specifier, the specifier is resolved prior to coercion. For example, if desc represents

the number of words of paragraph 3 of document "foo"

and type is typeIEEE32BitFloatingPoint, then @ desc is resolved (yielding the integer count of words), then coerced into a floating-point value.

Returns
An OS status code.
Exceptions
none
Parameters
descThe Apple Event descriptor.
typeThe destination type of the coercion.
dataThe destination address.
sizeThe destination's size.
