Detailed Description

template<UInt32 CLASS, UInt32 KIND>
class B::Event< CLASS, KIND >

Template class representing a specific Carbon Event.

Usually this template is specialised in order to add member variables holding event parameters, or to override Update() and/or Retrieve(). However, if a Carbon Event has no input or output parameters, this class may be instantiated directly.
