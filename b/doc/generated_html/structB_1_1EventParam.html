Detailed Description

template<EventParamName NAME>
struct B::EventParam< NAME >

Gets & sets a specific Carbon Event parameter in a type-safe manner.

EventParam utilises EventParamNameTrait to determine the parameter type. The parameter type is then used in turn to determine the language type.

This class is basically for convenience. It allows you replace:

EventParamAndType<NAME, TYPE>

with

EventParam<NAME>

See EventParamAndType and EventParamNameTrait for more information.
