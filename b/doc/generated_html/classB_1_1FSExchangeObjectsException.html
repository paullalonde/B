Detailed Description

FSExchangeObjects exception class.

Objects of this class are thrown when FSExchangeObjectsCompat() fails. It contains the source and destination FSRefs, which contain the identities of the two files to be exchanged at the time of failure.

In all instances of this class, what() returns "B::FSExchangeObjectsException".
