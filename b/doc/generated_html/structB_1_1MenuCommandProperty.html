Detailed Description

template<OSType SIG, OSType TAG, typename T>
struct B::MenuCommandProperty< SIG, TAG, T >

Gets & sets a menu command property in a type-safe manner.

MenuCommandProperty allows one to access a specific property (identified by the SIG and TAG template parameters) of a MenuRef/MenuCommand pair in a type-safe manner (this safety is provided indirectly by the T template parameter).

The idea is to allow for easy definition of properties in a type-safe manner. For example, if a client of a MenuRef needs to associate a URL with a menu command, it can simply declare a typedef for it:

typedef B::MenuCommandProperty<kMySig, kMyURLTag, CFURLRef> UrlMenuCommandProperty;

The client can then get & set the property:

B::Url myUrl;
myUrl = UrlMenuCommandProperty::Get(kMyCmd);
UrlMenuCommandProperty::Set(kMyCmd, myUrl);
