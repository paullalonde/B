Detailed Description

Wrapper around Carbon Ptr type.

This class provides "resource acquisition is initialization" (RAII) semantics for Carbon Ptrs. It features a destructive copy constructor and assignment operator similar in spirit to std::auto_ptr.

Note
This class deals with the raw Ptr type, so it isn't type-safe.
