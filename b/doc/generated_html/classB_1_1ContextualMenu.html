MenuItemIndex B::ContextualMenu::GetFirstItem ( )
protected

Returns the first menu item after the help item, if any.

Determining the first "real" item in a contextual menu can be tricky, because passing kCMHelpItemRemoveHelp will remove the help item entirely. So we follow this heuristic: if the first menu item has a command ID, it's taken to be the first item; else, the first item is the one following the first separator line.

This depends on the fact that the Menu Manager doesn't attach a command ID when it prepends a "help" item onto a contextual menu.
