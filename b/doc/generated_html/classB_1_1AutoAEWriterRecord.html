Detailed Description

A stack-based class for opening a descriptor record.

The constructor opens the descriptor record, the destructor closes it.

B::AutoAEWriterRecord::AutoAEWriterRecord ( AEWriter &  writer)

Opens a descriptor record.

Use this function when writing an element of a descriptor record.

If the stream itself isn't already open, it is opened via Open().
