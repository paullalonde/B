Detailed Description

Acts as an accumulator for menu item or control state changes.

Instances of this class are passed to Carbon Event handlers for the kEventClassCommand/kEventCommandUpdateStatus event. The handlers should call appropriate member functions to indicate a menu item's enabled status, text, mark character, etc. When the handler returns, the menu item is updated to reflect the new state.
