Detailed Description

template<EventParamName NAME, EventParamType TYPE>
struct B::EventParamAndType< NAME, TYPE >

Gets & sets a specific Carbon Event parameter of a specific type in a type-safe manner.

EventParamAndType utilises EventParamTypeTrait<TYPE> to actually retrieve & store the data. See EventParamTypeTrait for more information.
