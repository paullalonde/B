Detailed Description

template<typename OBJ_SPEC, typename OBJ_CLASS, typename RNG_OBJ1, typename RNG_OBJ2>
class B::AEEventFunctor::RangeSpecifier< OBJ_SPEC, OBJ_CLASS, RNG_OBJ1, RNG_OBJ2 >

Functor that holds an object specifier for a comparison.

This functor is meant to be passed to any of the arguments to instantiations of AEEvent that take an object specifier.
