#define BLocalizedString (   key,
  comment 
)    B::Bundle::Main().GetLocalisedString((key), (key), NULL)

Retrieves a localised string residing in the main bundle's Localizable.strings file.

Parameters
keyA CFStringRef containing the name of the string to retrieve.
commentA description of the string's purpose. Although this description is not compiled into the application's code, it is extracted by the genstrings tool and placed into .strings files as an aid to translators.
Returns
A B::String containing the value of key, or key itself if its value wasn't found.
