Detailed Description

template<OSType SIG, OSType TAG, typename T>
struct B::MenuItemProperty< SIG, TAG, T >

Gets & sets a menu item property in a type-safe manner.

MenuItemProperty allows one to access a specific property (identified by the SIG and TAG template parameters) of a MenuRef/MenuItemIndex pair in a type-safe manner (this safety is provided indirectly by the T template parameter).

The idea is to allow for easy definition of properties in a type-safe manner. For example, if a client of a MenuRef needs to associate a URL with a menu item, it can simply declare a typedef for it:

typedef B::MenuItemProperty<kMySig, kMyURLTag, CFURLRef> UrlMenuItemProperty;

The client can then get & set the property:

MenuRef myMenu;
B::Url myUrl;
myUrl = UrlMenuItemProperty::Get(myMenu, 1);
UrlMenuItemProperty::Set(myMenu, 2, myUrl);
