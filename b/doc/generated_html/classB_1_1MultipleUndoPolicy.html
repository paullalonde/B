Detailed Description

Multiple undo policy.

This is a full implementation of UNDO_POLICY. It supports an arbitrary number of undos and redos in two stacks. Nested open undo groups are allowed.

This class can be used as a template parameter to those classes that take an UNDO_POLICY template parameter.

Currently, two classes take undo policies as template parameters: Application and Document.
