Detailed Description

Note
We don't do the following controls, since they don't support compositing windows:
  • TE-based edit text control
  • list box control
  • scroll text box control
When creating a view programatically (ie via XXX::Create()), the view is initially made invisible. This paves over an inconsistency between the older Control-Mgr-style views and the new HIView-style ones.
