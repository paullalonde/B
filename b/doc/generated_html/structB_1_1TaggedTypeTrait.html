Detailed Description

template<OSType TYPE>
struct B::TaggedTypeTrait< TYPE >

Maps an integral constant onto a C/C++ language type.

TaggedTypeTrait maps an integer constant identifying a Carbon Event parameter type (EventParamType) or an Apple Event descriptor type (DescType) onto a C/C++ language type. Both Carbon Events and AppleEvents use the same namespace for argument types, which is why they both use this common traits class.

The idea is that the language type corresponding to a given parameter type can be determined via this expression:

TaggedTypeTrait<TYPE>::Type

For example, in the definition:

TaggedTypeTrait<typeBoolean>::Type my_var;

my_var is of type bool.

Note that the generic definition of TaggedTypeTrait is deliberately useless, because TaggedTypeTrait is meant to be specialised for each parameter type.
