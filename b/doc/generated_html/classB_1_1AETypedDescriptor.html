Detailed Description

template<DescType TYPE>
class B::AETypedDescriptor< TYPE >

Type-safe wrapper class around an Apple Event descriptor.

This class provides a simple type-safe way of controlling the lifetime of an Apple Event descriptor. Its constructors allocate the descriptor, and its destructor releases it. It's really that simple.
