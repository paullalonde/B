Detailed Description

Functor for sending Apple Events to an arbitrary (Carbon or Classic) process.

This class is suitable for use as the TARGET parameter of AEWriter's constructor. You can target any app that has a Process Manager connection; usually that means Classic, Carbon and Cocoa (but not BSD-style) applications.
