Detailed Description

template<typename T>
class B::AutoTypedResourceHandle< T >

Type-safe wrapper around Carbon resource handles.

This class provides "resource acquisition is initialization" (RAII) semantics for Carbon resource handles. It features a destructive copy constructor and assignment operator similar in spirit to std::auto_ptr.

It's assumed that the resource handles given to objects of this type match the template parameter T. T can be any POD (aka Plain Old Data) type.
