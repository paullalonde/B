Create an object and its HIToolbox view, from the given parameters.

Note
The view is initially invisible.
Parameters
inViewIDThe new view's ID; may be zero in which case the view doesn't get an HIViewID.
inSuperviewThe new view's superview; may be NULL, in which case the view is "standalone".
inFrameThe new view's frame; may be NULL, in which case the view doesn't get a frame with particular values. If inSuperview is not NULL, the frame is assumed to be in inSuperview's coordinates.
inFromNibThe nib from which to load other resources; may be NULL.
inFactoryThe factory to use for instantiating the derivative of View;; may be NULL, in which case the default view factory is used instead.
