This macro determines if any Tiger-specific declarations are accessed in the B source.

There appears to be no officially sanctioned way of telling what version of the OS (OS SDK actually) we are building with. Fortunately we can rely on a hack: we test for the definition of a symbol which only exists on Panther. If it exists, we're building on Panther or later; else, we're building on Jaguar.
