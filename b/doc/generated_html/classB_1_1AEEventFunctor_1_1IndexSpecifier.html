Detailed Description

template<typename OBJ_SPEC, typename OBJ_CLASS, typename INDEX>
class B::AEEventFunctor::IndexSpecifier< OBJ_SPEC, OBJ_CLASS, INDEX >

Functor that holds an object specifier with key form formAbsolutePosition.

This functor is meant to be passed to any of the arguments to instantiations of AEEvent that take an object specifier.

Three pieces of information are required to generate an object specifier with this key form: the containing object, the desired class of one of its elements, and the index of that element.

Accordingly, this class is defined in terms of three template parameters, OBJ_SPEC, OBJ_CLASS and INDEX, which must be functors capable of writing out the corresponding elements to the Apple Event record.
