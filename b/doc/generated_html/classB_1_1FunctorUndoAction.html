Detailed Description

An undoable action that delegates to functor.

This convenience class makes it easy to build an undoable action out of any object convertible to a boost::function; this includes objects generated by boost::bind, meaning that one can use boost::bind to pass member functions to FunctorUndoAction's constructor.

The functor must take no arguments and return no result. If it needs to contain state, boost::bind may be used to bind state to the functor.

Instances of FunctorUndoAction are passed to implementations of UNDO_POLICY and are maintained by them until an Undo or Redo action is performed.
