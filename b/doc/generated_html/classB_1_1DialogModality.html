Detailed Description

Helper object representing a dialog's modality.

This class encapsulates a WindowModality / WindowRef pair, and may be used to specify a dialog's modality. Dialog modality is what distinguishes a modeless dialog from an application-modal dialog from a sheet. This class is used by Alert, Dialog, and NavDialog.

You will typically construct a DialogModality object in one of two ways:

  1. If you know which modality you want, call one of the static instantiation functions. For example, Modal() will return an object specifying application-modal modality.
  2. If you get a WindowModality and possibly a WindowRef handed to you by someone else, use DialogModality's constructor directly.
Note
This class doesn't support kWindowModalitySystemModal. Mac OS X maps system-modal dialogs onto application-modal dialogs anyway, so it's not a problem.
