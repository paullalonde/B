Detailed Description

Validation-failed functor that beeps.

This is a simple helper class that is meant to be passed to TextFieldPart::ConnectValidationFailed(). It beeps when invoked. This has the effect of causing a beep whenever the user enters a character that is rejected by one of the filters attached to the text field part.
