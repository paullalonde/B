Detailed Description

Exception helper class.

This class can be embedded within an exceptions object as a member variable. It holds a CFStringRef, which is converted on-demand into a const char* via GetString().

The class' member functions are set up so that they cannot throw at an inopportune time (i.e. during construction). Its throw-specs are consistent with those of std:exception and its derived classes.
