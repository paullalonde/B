Detailed Description

Functor for writing Apple Event keywords to an AEWriter.

Encapsulates an Apple Event keyword for operator << . In the following example, an Apple Event descriptor whose data is given by my_value is written to the stream with a keyword of kMyKeyword :

AEWriter writer(...);
writer << AEKey(kMyKeyword) << myValue;
Note
The keyword isn't written immediately to the AEWriter, but rather stored for later. It actually gets written when descriptor data is written, or when a descriptor/record/list is opened.
