Detailed Description

template<class WINDOW>
class B::AutoWindow< WINDOW >

Manages a "stack-based" window.

AutoWindow's constructor instantiates a window, and its destructor disposes of it. It is most useful for displaying modal dialogs. For example:

void foo(Nib& nib, AEObject* container)
{
AutoWindow<B::Window> autoWin(nib, "my window", container);
// Do stuff here. The window gets disposed when we leave
// this scope.
}
