Detailed Description

template<typename PROC, typename UPP>
class B::AutoDefUPP< PROC, UPP >

Pseudo Universal Procedure Pointer (UPP) wrapper.

Used instead of AutoUPP UPPs are not really allocated or deallocated, but merely type cast from their corresponding ProcPtr.
