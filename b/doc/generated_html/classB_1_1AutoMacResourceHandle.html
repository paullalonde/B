Detailed Description

Wrapper around Carbon resource handles.

This class provides "resource acquisition is initialization" (RAII) semantics for Carbon resource handles. It features a destructive copy constructor and assignment operator similar in spirit to std::auto_ptr.

Note
This class deals with the raw Handle type, so it isn't type-safe.
