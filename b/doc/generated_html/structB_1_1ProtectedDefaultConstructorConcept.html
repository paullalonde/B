Detailed Description

template<class TT>
struct B::ProtectedDefaultConstructorConcept< TT >

All descendants of NavDialog need a default constructor; however, to prevent accidental mis-use (by declaring a NavDialog-derived object on the stack, for example), the constructor must be protected. This concept check verifies that condition indirectly.
