Detailed Description

Holds a set of Carbon Event handlers for a given Carbon Event target.

EventHandler holds a set of mappings between Carbon Event class/kind pairs on the one hand, and functions implementing the Carbon Events on the other.

Often, clients will want to handle a Carbon Event via a member function. This it can be done by one of the overloads of Add() which takes an object and a pointer-to-member-function for that object. For example:

class MyObject {
bool MyHandleEvent(Event<kMyEventClass, kMyEventKind>& event);
};
MyObject obj;
myEventHandler.Add(&obj, &MyObject::MyHandleEvent);
