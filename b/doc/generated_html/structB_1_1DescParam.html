Detailed Description

template<DescType TYPE>
struct B::DescParam< TYPE >

Reads & writes an Apple Event descriptor in a type-safe manner.

DescParam provides a type-safe wrapper around the Apple Event Manager calls that read and write Apple Event descriptors.

The class's type-safety is provided by the related type TaggedTypeTrait, which maps an DescType onto a C/C++ language type. Thus DescParam augments TaggedTypeTrait with Apple Event accessors.
