Detailed Description

Property list creation exception class.

Objects of this class are thrown when a property list creation function such as CFPropertyListCreateFromXMLData() fails. The exception holds the error message (a CFStringRef) returned by the failing function.

In all instances of this class, what() returns "B::PropertyListCreateException".
