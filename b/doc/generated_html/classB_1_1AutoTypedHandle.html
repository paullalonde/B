Detailed Description

template<typename T>
class B::AutoTypedHandle< T >

Type-safe wrapper around Carbon Handle type.

This class provides "resource acquisition is initialization" (RAII) semantics for Carbon Handles. It features a destructive copy constructor and assignment operator similar in spirit to std::auto_ptr.

It's assumed that the handles given to objects of this type match the template parameter T. T can be any POD (aka Plain Old Data) type.
