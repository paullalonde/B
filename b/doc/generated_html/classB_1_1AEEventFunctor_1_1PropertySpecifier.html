Detailed Description

template<typename OBJ_SPEC, typename PROP_ID>
class B::AEEventFunctor::PropertySpecifier< OBJ_SPEC, PROP_ID >

Functor that holds an object specifier for a property of an object.

This functor is meant to be passed to any of the arguments to instantiations of AEEvent that take an object specifier.

Two pieces of information are required to generate an object specifier with this key form: the object, the desired property within it. Accordingly, this class is defined in terms of two template parameters, OBJ_SPEC and PROP_ID, which must be functors capable of writing out the corresponding elements to the Apple Event record.
