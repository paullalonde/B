Detailed Description

template<class DOC_TYPE, DescType OSA_CLASS = cDocument>
class B::SingleDocumentFactory< DOC_TYPE, OSA_CLASS >

A document factory for applications that only support a single document type.

Use it as the DOC_FACTORY residing within your application's DOC_POLICY.

Since the document's type is predetermined, it is specified in the DOC_TYPE and OSA_CLASS template parameters.

See also
DOC_FACTORY.
