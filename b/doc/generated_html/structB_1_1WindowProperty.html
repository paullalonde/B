Detailed Description

template<OSType SIG, OSType TAG, typename T>
struct B::WindowProperty< SIG, TAG, T >

Gets & sets a window property in a type-safe manner.

WindowProperty allows one to access a specific property (identified by the SIG and TAG template parameters) of an WindowRef in a type-safe manner (this safety is provided indirectly by the T template parameter).

The idea is to allow for easy definition of properties in a type-safe manner. For example, if a client of a WindowRef needs to associate a URL with a window, it can simply declare a typedef for it:

typedef B::WindowProperty<kMySig, kMyURLTag, CFURLRef> UrlWindowProperty;

The client can then get & set the property:

WindowRef myWindow1, myWindow2;
B::Url myUrl;
myUrl = UrlWindowProperty::Get(myWindow1);
UrlWindowProperty::Set(myWindow2, myUrl);
