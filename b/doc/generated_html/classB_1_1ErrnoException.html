Detailed Description

BSD errno exception class.

This class contains an OS-defined error code in the namespace used by the errno global variable. Possible values are ENOMEM, EACCESS, etc. Typically an exception of this class is thrown after a BSD-level function returns a non-zero result code.

In all instances of this class, what() returns "B::ErrnoException".
