Detailed Description

template<typename T>
struct B::ViewDataTrait< T >

Gets & sets HIVieRef data in a type-safe manner.

ViewDataTrait provides a type-safe wrapper around the Control Manager calls GetControlData() and SetControlData().

The template parameter, T, is a C/C++ language type. It's the type of the data to be stored into or retrieved from the view.
