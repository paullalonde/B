Detailed Description

template<OSType SIG, OSType TAG, class OBJTR>
struct B::ToolboxObjectPropertyBase< SIG, TAG, OBJTR >

Helper class that serves as the common ancestor of instantiations of ToolboxObjectProperty.

ToolboxObjectProperty needs to call some functions in the toolbox object which are independent of any property data type, so in the interest of saving a bit of space (as well as reducing the work required to specialise ToolboxObjectProperty), those functions have been factored out into this little helper class.
