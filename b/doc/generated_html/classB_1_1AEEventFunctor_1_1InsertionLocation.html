Detailed Description

template<typename OBJ_SPEC, typename POSITION>
class B::AEEventFunctor::InsertionLocation< OBJ_SPEC, POSITION >

Functor that holds an insertion location.

This functor is meant to be passed as the inInsertionLoc argument to AEEvent<kAECoreSuite, kAEClone>::Send(), AEEvent<kAECoreSuite, kAECreateElement>::Send(), and AEEvent<kAECoreSuite, kAEMove>::Send().

An insertion location is an Apple Event record containing two elements: an object specifier, and an enumerated value indicating a position relative to the object specifier.

Accordingly, this class is defined in terms of two template parameters, OBJ_SPEC and POSITION, which must be functors capable of writing out the corresponding elements to the Apple Event record.
